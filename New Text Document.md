# Bone Fracture Detection using YOLOv8
 
## Problem
Detect different types of bone fractures from X-ray images using object detection.
 
## Dataset
- ~16.5k images
- Classes:
  - Elbow positive
  - Fingers positive
  - Forearm fracture
  - Humerus fracture
  - Shoulder fracture
  - Wrist fracture

- Model: YOLOv8

- Evaluation: mAP, precision, recall
